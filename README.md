## Instalação

1. Clone o repositório:

    ```bash
    git clone https://github.com/leofrs/psf
    ```

2. Instale as dependências:

    ```bash
    npm install
    ```

3. Inicie o servidor:

    ```bash
    npm run dev
    ```
